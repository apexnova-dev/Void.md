# Reference / Spec Code (Not Bundled)

The `.js` files in this directory are **reference implementations** and spec-aligned code for the planned v1.2.0 enhanced notes/comments system (Tiptap editor, slash commands, threading). They are **not** part of the current shipped application.

The main app is a single file: **`task-manager.html`** at the repository root. It does not load these files. Use them as design reference or a starting point when implementing the architecture described in [ARCHITECTURE_SPECIFICATION_v1.2.0.md](../ARCHITECTURE_SPECIFICATION_v1.2.0.md).
