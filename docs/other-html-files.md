# Other HTML Files in This Repository

This document describes HTML files in the repository that are **not** the main application.

| File | Purpose |
|------|---------|
| **void.html** | **Main application.** This is the file to open and use. |
| **MDTM.html** | Alternate/legacy build of the application (older styling, no dark mode hierarchy). Kept for reference. |
| **task-manager_brokenFile.html** | Backup or debug snapshot of the application from a past session. Not for production use. |
| **test-page.html** | Test harness that loads `void.html` in an iframe with console capture. Used for manual testing; see [AGENTS.md](../AGENTS.md) for how to run tests. |

Use **void.html** for normal use. The others are for development, testing, or reference only.
